# 二年级数学出题系统

## 📚 项目简介

这是一个专为小学二年级设计的数学出题系统，支持多种题型的自动生成，帮助老师快速制作练习题和测试卷。

## ✨ 主要功能

### 1. 多种题型支持
- 🧠 **口算练习**：100以内加减法，9以内乘法（一行四题，8行×4列=32题）
- 🔢 **竖式练习**：带进位/退位的加减法竖式计算
- 📝 **基础算式理解**：加法、减法、乘法的基本概念
- 🔢 **乘法口诀**：多种题型（填空、选择、应用）
- ➕ **进位退位分析**：理解进位和退位的规律
- 🔍 **看错题分析**：培养检查和纠错能力
- 📏 **画线题**：长度测量和作图练习
- 💰 **购物问题**：实际生活场景应用题
- 🧭 **方向识别**：空间方位判断

### 2. 智能功能
- ✅ 一键显示/隐藏答案
- 📖 详细解题过程展示
- 📥 PDF格式下载（隐藏答案）
- 📅 日历计划表下载（多种风格）
- 🔄 题型数量自定义
- 🎲 随机生成题目

## 🎯 特色亮点

1. **题型丰富**：覆盖二年级数学主要知识点
2. **界面友好**：简洁直观的操作界面
3. **打印优化**：适合A4纸打印的排版设计
4. **灵活配置**：点击增加题型数量，支持混合出题
5. **即时反馈**：可选择性显示答案和详解

## 📖 使用说明

### 生成题目
1. 点击题型标签增加题目数量
2. 点击"生成题目"按钮
3. 题目将自动显示在页面中

### 查看答案
- 点击"显示答案"按钮查看所有题目答案
- 再次点击可隐藏答案

### 查看详解
- 点击"显示详解"查看每道题的详细解答过程
- 帮助学生理解解题方法

### 下载PDF
- 点击"下载PDF"按钮
- 自动生成不含答案的练习题PDF文件
- 适合直接打印使用

### 下载日历
1. 选择喜欢的日历模板
2. 点击"下载日历"按钮
3. 获得2025年精美日历计划表
